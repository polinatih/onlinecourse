Как работает Facade
🏛 Facade

EnrollmentFacade.enrollStudent() скрывает за собой всю цепочку действий:

Шаг 1: Загрузить пользователя и курс из репозиториев
Шаг 2: Проверить, не записан ли уже студент
Шаг 3: Сохранить запись в БД через UserRepository
Шаг 4: Синхронизировать с LMS-системами (Adapter — Moodle, Google Classroom)
Шаг 5: Вернуть список доступных тестов курса
Контроллер вызывает один метод фасада — не знает о деталях подсистем.
